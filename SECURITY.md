# Security Policy

## Scope

This repository contains documentation source files only. There is no executable code or application logic.

Security concerns related to the decode-orc or encode-orc applications themselves should be reported to the main project repository.

## Reporting a vulnerability in the main project

Please do **not** open public issues for suspected security vulnerabilities in decode-orc or encode-orc.

Use one of the following:

- GitHub Security Advisories (preferred): https://github.com/simoninns/decode-orc/security/advisories/new
- Maintainer contact: https://github.com/simoninns

Include:

- A clear description of the issue
- Steps to reproduce (if available)
- Affected versions or commit range
- Potential impact

## Reporting issues with this documentation repository

If you believe the documentation itself contains a security concern (for example, instructions that could lead users to unsafe configurations), please open an issue:

https://github.com/simoninns/decode-orc-docs/issues
