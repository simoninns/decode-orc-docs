# Testcards

Testcard assets are used by encode-orc when encoding TBC video and a variety of input formats are supported.

The testcard assets live under `testcard-images/` in the repository:

- `pal-png/` — PAL PNG images (active-picture sized)
- `ntsc-png/` — NTSC PNG images (active-picture sized)
- `pal-raw/` — PAL raw Y'CbCr 4:2:2 test frames
- `ntsc-raw/` — NTSC raw Y'CbCr 4:2:2 test frames
- `pal-mov/` — PAL QuickTime MOV sources
- `ntsc-mov/` — NTSC QuickTime MOV sources
- `pal-mp4/` — PAL MP4 sources
- `ntsc-mp4/` — NTSC MP4 sources
- `pal-wrwetzel-png/` — PAL PNG testcards by Bill Wetzel
- `ntsc-wrwetzel-png/` — NTSC PNG testcards by Bill Wetzel
- `sound/` — Audio assets used by sound-related tests
