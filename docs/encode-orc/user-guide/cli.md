# Command Line Usage

```bash
# Basic usage
./encode-orc project.yaml

Note: The specification of all input files and output targets is contained in the YAML project file.

# Verbose logging
./encode-orc project.yaml --log-level debug --log-file /tmp/encore-orc.log

# Help information
./encode-orc --help

# Version information
./encode-orc --version
```
